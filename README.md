##  目前支持功能

|              |                                                              |
| ------------ | ------------------------------------------------------------ |
| 功能         | 备注                                                         |
| 女仆         | 需要提前把需要购买的物品设置好                               |
| 领取邮件     |                                                              |
| 帮派圣兽     | 需要提前设置好阵容 会打五次并领钻石                          |
| 每日福利     |                                                              |
| 每日跨服竞技 | 周三到周日执行,只领10次奖励,需要周一周二手动把拍名打上去     |
| 天命棋局     | 需要提前把设置的三个勾勾上.只能自动打5次并领5次奖励,继续打需要手动 |

## 使用方法

### 下载

下载`images.zip`和`GameAssitant.zip`两个压缩包 并放在合适的位置并分别解压缩

### 配置

GameAssitant文件夹中找到`config.json`文件并用记事本等文本工具打开

![image-20250106221138052](https://pic-migrate.oss-cn-guangzhou.aliyuncs.com/202501062224818.png)

#### 修改内容

引号内部改成**你本地的**`images文件夹`路径(就是第1步中解压images文件夹的位置),并保存文本

![image-20250106221432614](https://pic-migrate.oss-cn-guangzhou.aliyuncs.com/202501062224027.png)

### 运行

这个`GameAssitant.exe`是要辅助程序,双击运行

![image-20250106221732464](https://pic-migrate.oss-cn-guangzhou.aliyuncs.com/202501062224831.png)

也可以右键给它新建一个快捷方式放到桌面

![image-20250106221926149](https://pic-migrate.oss-cn-guangzhou.aliyuncs.com/202501062224835.png)

### 运行游戏

![image-20250106222135604](https://pic-migrate.oss-cn-guangzhou.aliyuncs.com/202501062224979.png)

目前仅支持单账号模式,需要登陆好进界面 把游戏内乱七八糟的弹窗都先点掉

点开始就可以了.其他的框是我测试用的 以后会挪走
